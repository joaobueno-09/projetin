O que é o seno?
Seno é a razão entre o cateto oposto e a hipotenusa.
O que é o cosseno?
Cosseno é a razão entre o cateto adjacente e a hipotenusa.
Qual é a fórmula da tangente?
Tangente é a razão entre o cateto oposto e o cateto adjacente.
O que é a identidade pitagórica?
a² + b² = c², onde a e b são os catetos e c é a hipotenusa.
Como calcular a hipotenusa no triângulo retângulo?
Usando a fórmula: c = √(a² + b²)
O que é a cotangente?
Cotangente é o inverso da tangente, ou seja, cot(θ) = 1/tan(θ).
O que é a secante?
Secante é o inverso do cosseno, ou seja, sec(θ) = 1/cos(θ).
O que é a cossecante?
Cossecante é o inverso do seno, ou seja, csc(θ) = 1/sin(θ).
Qual é a fórmula de Euler?
e^(iπ) + 1 = 0, uma das fórmulas mais famosas da matemática.
